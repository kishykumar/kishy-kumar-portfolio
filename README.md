# Kishy Kumar Portfolio

Modern, minimalist portfolio website for Kishy Kumar - Director of Engineering at Oracle.

## Features

- Clean, modern design inspired by top YC companies (Linear, Raycast)
- Dark theme with gradient accents
- Fully responsive layout
- Smooth scroll animations
- Comprehensive bio covering experience, skills, and accomplishments
- Focus on technical expertise and leadership

## Content

- **Hero**: Overview with Oracle leadership, JSON-Relational Duality innovation
- **About**: Personal background and engineering philosophy
- **Experience**: Timeline from Engineer to Director at Oracle
- **Key Work**: Deep dive on patents, publications, and major projects
- **Skills**: Technical and leadership competencies
- **Accomplishments**: Patent, SIGMOD paper, conference committee roles
- **Contact**: Links to LinkedIn and email

## Tech Stack

- HTML5 + CSS3
- Vanilla JavaScript (no frameworks needed for performance)
- Google Fonts (Inter)
- CSS Grid/Flexbox for responsive layout
- CSS custom properties for theming

## Local Development

```bash
# Simply open in a browser
open index.html
```

Or use a simple HTTP server:
```bash
# Using Python
python3 -m http.server 3000

# Using Node.js
npx serve -p 3000
```

## Deployment

This is a static site. Deploy to any static hosting:

### Vercel / Netlify
1. Drag and drop the `kishy-kumar-portfolio` folder
2. Or connect a GitHub repository

### GitHub Pages
1. Push to a GitHub repo
2. Enable Pages in repository settings
3. Set source to `main` branch, folder `/ (root)`

### Free hosting options (as requested):
- **Vercel**: vercel.com (free tier, custom domain included)
- **Netlify**: netlify.com (free tier, custom domain included)
- **Cloudflare Pages**: pages.cloudflare.com (free tier)

## Domain Options

For a completely free ".tk" domain as requested:
1. Register at Freenom.com (search for `kishykumar.tk`)
2. Deploy site to Vercel/Netlify
3. Add custom domain in hosting settings
4. Update nameservers or DNS records at Freenom

## Customization

- Update personal links in `index.html` (LinkedIn, email)
- Adjust bio content to match your experience
- Colors can be modified in `styles.css` CSS variables
